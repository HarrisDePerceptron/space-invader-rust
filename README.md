## Space invader meets Rust 
![Invader](assets/invader_v1.gif)


### Installation

Install rust if not already available

```
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```
### Getting started

```
cargo run
```
